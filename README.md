# Sake

Snake is the game that give new life of old Snake game from Nokia.

  - 1 player mode : get as many food as you can
  - 2 player mode : grab as many food to wait other Snake hit your boy.

```sh
How to control 
Player 1: W,A,S,D
Player 2: UP, Left, Down, Right
Add second player
Reset Game: F5
```
